# Installation
1. Download the latest version of addon from <a href="https://github.com/AkaSokuro/NFSHPR-DiscordRPC-Rework/releases">Releases</a>. (<b>Choose `NFSHPR-DiscordRPC-Rework.zip` not Source code</b>)
2. Unzip the file into the game directory, where the executable (.exe) is located. (<b>Overwrite existing files</b>)
3. Launch the game to test it.

# Uninstallation
### Full Uninstallation:
- Delete `scripts` and `dinput8.dll` from the game directory.

### Only This Addon:
- Delete `NFSHPR DiscordRPC Rework.asi` in the `scripts` folder

# Build Project
In order to build the project yourself, you have to make your own SDK for the game, I don't release the SDK because of the many vulnerabilities for cheaters to use it to gain advantage against other players.<br><br>
<b>Please respect others and have fun.</b>
